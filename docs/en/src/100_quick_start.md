# Quick Start

This guide will help you get MSD-RS up and running quickly.

The usual workflow is:

1. [Installation](./101_installation.md): Build the project or use the Docker image.
2. [Server](./102_server.md): Start the MSD server.
3. [Client](./103_client.md): Connect to the server using the shell or other clients.
4. [SQL Interface](./104_sql_interface.md): Interact with the data using SQL.
