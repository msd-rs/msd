# API 参考

## 1. HTTP API

### 1.1 POST /query — 查询数据

**Content-Type**: `text/plain`

**请求体**: SQL 语句（纯文本）

**响应格式**:
- 默认：`application/x-ndjson`（每行一个 JSON 对象）
- MSD Client：`application/x-msd-table-frame`（二进制帧流）

**示例**：

```bash
# 查询表结构
curl -X POST http://127.0.0.1:50510/query \
  -H "Content-Type: text/plain" \
  -d "DESCRIBE kline"

# 查询数据
curl -X POST http://127.0.0.1:50510/query \
  -H "Content-Type: text/plain" \
  -d "SELECT open, close FROM kline WHERE obj = 'SH600519' AND ts >= 2024-01-01 LIMIT 10"
```

**NDJSON 响应示例**：
```json
{"table":"kline","obj":"SH600519","columns":[{"name":"ts","kind":"DateTime"},{"name":"open","kind":"Float64"}],"rows":[[1704067200000000,1650.0],[1704153600000000,1660.0]]}
```

### 1.2 PUT /table/{table_name} — 导入数据

**Content-Type**: `text/csv` 或 `application/x-msd-table-frame`

**URL 参数**: `?skip=N`（跳过 CSV 前 N 行，如跳过 header）

**CSV 格式要求**:
- 第一列必须为 `obj`（对象名称）
- 后续列对应表结构中的列（按顺序）
- 时间戳列值需为微秒时间戳（整数）或可解析的日期字符串

**示例**：

```bash
# CSV 导入（跳过 header 行）
curl -X PUT "http://127.0.0.1:50510/table/kline?skip=1" \
  -H "Content-Type: text/csv" \
  --data-binary @data.csv

# 二进制格式导入
curl -X PUT http://127.0.0.1:50510/table/kline \
  -H "Content-Type: application/x-msd-table-frame" \
  --data-binary @data.bin
```

**CSV 数据示例**：
```csv
obj,ts,open,high,low,close,volume
SH600519,1704067200000000,1650.0,1660.0,1645.0,1655.0,1000000
SH600519,1704153600000000,1660.0,1670.0,1655.0,1665.0,1200000
```

### 1.3 GET /ws — WebSocket 连接

**功能**：实时订阅表更新通知。

**消息格式**（JSON）：

**客户端 -> 服务器**：
```json
// 订阅
{"type": "Subscribe", "table": "kline", "obj": "*"}

// 取消订阅
{"type": "Unsubscribe", "table": "kline", "obj": "SH600519"}
```

**服务器 -> 客户端**：
```json
// 通知（有数据更新）
{"type": "Notify", "table": "kline", "obj": "SH600519", "count": 1}

// 状态
{"type": "Status", "status": "ok"}
```

### 1.4 GET /mcp — MCP 服务

**功能**：提供 AI 可调用的数据目录和结构信息。

**支持的工具**：
- `list_tables`：列出所有表名
- `get_table`：获取指定表的结构信息

## 2. TableFrame 二进制协议

### 2.1 帧格式

```
Offset  Size  Field
------  ----  -----
0       2     Magic:  0x4D 0x53 ("MS")
2       2     Version: 0x00 0x01 (v1)
4       4     Frame Size: 总帧长度 (Big Endian u32)
8       N     Payload: bincode 序列化的 Table
8+N     4     CRC32: 对 Payload 的校验和 (Big Endian u32)
```

### 2.2 Table 序列化格式

Table 使用 bincode 序列化，包含完整的列定义和列数据。

### 2.3 Rust 实现

```rust
use msd_request::table_frame;

// 打包
let frames: Vec<u8> = table_frame::pack_table_frame(&table)?;

// 解包
let table: Table = table_frame::unpack_table_frame(&frames)?;
```

### 2.4 Python 实现

```python
import msd

# 自动使用 TableFrame 协议
tables = msd.query("http://127.0.0.1:50510", "SELECT * FROM kline LIMIT 100")
for obj_name, df in tables:
    print(f"Object: {obj_name}")
    print(df)
```

## 3. SQL 参考

### 3.1 数据查询 (SELECT)

```sql
-- 基本查询
SELECT * FROM table_name;

-- 指定列
SELECT col1, col2 FROM table_name;

-- 按对象过滤
SELECT * FROM table_name WHERE obj = 'SH600519';

-- 按时间范围过滤
SELECT * FROM table_name WHERE ts >= 2024-01-01 AND ts < 2024-02-01;

-- 排序和限制
SELECT * FROM table_name ORDER BY ts DESC LIMIT 100;

-- 组合查询
SELECT open, high, low, close
FROM kline
WHERE obj = 'SH600519'
  AND ts >= 2024-01-01
  AND ts < 2024-06-01
ORDER BY ts ASC
LIMIT 1000;
```

### 3.2 创建表 (CREATE TABLE)

```sql
-- 基本创建
CREATE TABLE table_name (
    ts DATETIME,
    col1 FLOAT64,
    col2 FLOAT64,
    col3 STRING,
    col4 INT64
);

-- 带元数据
CREATE TABLE kline (
    ts DATETIME,
    open FLOAT64 agg=first desc='开盘价',
    high FLOAT64 agg=max desc='最高价',
    low FLOAT64 agg=min desc='最低价',
    close FLOAT64 agg=first desc='收盘价',
    volume FLOAT64 agg=sum desc='成交量'
) WITH (
    round='1d',
    chunkSize=200
);

-- 带链更新
CREATE TABLE snapshot (
    ts DATETIME,
    open FLOAT64,
    high FLOAT64,
    low FLOAT64,
    close FLOAT64
) WITH (
    chan='kline:changed_if(open,close),changed_if(high,close),changed_if(low,close),close'
);
```

**支持的列类型**：
- `DATETIME` — 日期时间（微秒时间戳）
- `INT32`, `INT64` — 有符号整数
- `UINT32`, `UINT64` — 无符号整数
- `FLOAT32`, `FLOAT64` — 浮点数
- `STRING` — 字符串
- `DECIMAL64`, `DECIMAL128` — 定点小数
- `BINARY` — 二进制数据
- `BOOL` — 布尔值

**支持的列元数据**：
- `agg=<method>` — 聚合方法（sum/count/min/max/avg/first/prev/uniq_count/diff_prev/diff_first）
- `desc='...'` — 字段描述

**支持的表元数据**：
- `round='<unit>'` — 时间聚合粒度（如 `1m`, `5m`, `1h`, `1d`）
- `chunkSize=<n>` — 每 chunk 行数
- `chan='<config>'` — 链更新配置
- `desc='...'` — 表描述
- `ignore_older_than=<timestamp>` — 忽略旧数据的时间戳阈值

### 3.3 插入数据 (INSERT)

```sql
-- VALUES 方式
INSERT INTO table_name VALUES
    ('obj1', ts1, val1, val2, ...),
    ('obj1', ts2, val3, val4, ...);

-- COPY 方式（CSV 字符串）
INSERT INTO table_name COPY
    obj,ts,col1,col2
    obj1,1704067200000000,1.0,2.0
    obj1,1704153600000000,1.5,2.5
    ;
```

### 3.4 删除数据 (DELETE)

```sql
-- 删除指定对象
DELETE FROM table_name WHERE obj = 'SH600519';

-- 删除时间范围
DELETE FROM table_name WHERE obj = 'SH600519' AND ts >= 2024-01-01 AND ts < 2024-02-01;
```

### 3.5 表管理

```sql
-- 查看表结构
DESCRIBE table_name;

-- 列出所有表
.tables

-- 删除表
DROP TABLE table_name;
```

### 3.6 注释 (COMMENT)

```sql
-- 为表添加描述
COMMENT ON TABLE table_name IS '表的描述信息';

-- 为列添加描述
COMMENT ON COLUMN table_name.column_name IS '列的描述信息';
```

## 4. Python SDK API

### 4.1 安装

```bash
pip install pymsd
```

### 4.2 快速开始

```python
from pymsd import create_msd_pandas, Client

# 创建客户端
client = Client("http://127.0.0.1:50510")

# 获取 pandas 格式的查询结果
pandas_query = create_msd_pandas()

# 查询
for obj, df in pandas_query("http://127.0.0.1:50510",
                              "SELECT * FROM kline WHERE obj='SH600519' LIMIT 10"):
    print(f"Object: {obj}")
    print(df)

# 使用高级客户端加载数据
kline_df = client.load("kline", obj="SH600519",
                        start="2024-01-01", end="2024-06-01")
```

### 4.3 主要 API

```python
# 同步查询
from pymsd import query
results = query(base_url, sql_statement)

# 异步查询
from pymsd import async_query
results = await async_query(base_url, sql_statement)

# CSV 导入
from pymsd import import_csv
import_csv(base_url, table_name, csv_data)

# DataFrame 导入
from pymsd import import_dataframes
import_dataframes(base_url, table_name, df_iterator)

# DataFrame 工厂
from pymsd import create_msd_pandas, create_msd_polars
pandas_query = create_msd_pandas()
polars_query = create_msd_polars()
```

### 4.4 Client 高级用法

```python
from pymsd.easy import Client

client = Client("http://127.0.0.1:50510")

# 基本加载
df = client.load("kline", obj="SH600519")

# 带时间范围
df = client.load("kline", obj="SH600519",
                  start="2024-01-01", end="2024-12-31")

# 指定字段
df = client.load("kline", obj="SH600519",
                  fields=["open", "high", "low", "close"])

# 连接多个表（基于 ts 和 obj）
df = client.load("kline", obj="SH600519") \
    .join("dividend", on="ts") \
    .join("shares", on="ts") \
    .get()
```

## 5. 认证

### 5.1 JWT Token 生成

```bash
# 生成 Token
msd token -a <secret-key> -r <role> -e <expire-days>

# 示例
msd token -a my-secret -r read -e 365
```

### 5.2 使用 Token

HTTP 请求头：
```
Authorization: Bearer <token>
```

本地请求（127.0.0.1）且未配置 auth-token 时不需要认证。

### 5.3 角色权限

| 角色 | 值 | 允许操作 |
|------|---|---------|
| read | 1 | SELECT, DESCRIBE |
| write | 2 | INSERT, DELETE, PUT /table/ |
| admin | 4 | CREATE TABLE, DROP TABLE, COMMENT |

权限可叠加（如 `-r admin` 拥有全部权限）。

## 6. 环境变量参考

| 变量 | 用途 | 默认值 |
|------|------|--------|
| `MSD_DB_PATH` | 数据库目录 | `./msd_db` |
| `MSD_LISTEN_ADDR` | 监听地址 | `127.0.0.1:50510` |
| `MSD_WORKERS` | Worker 线程数 | `8` |
| `MSD_AUTH_TOKEN` | JWT 密钥 | 无（关闭认证） |
| `MSD_TZ` | 时区偏移 | 本地时区 |
| `MSD_LOG_PATH` | 日志目录 | stdout |
| `MSD_SERVER_URL` | Shell 默认服务器地址 | `http://127.0.0.1:50510` |
| `MSD_PID_FILE` | PID 文件路径 | 无 |
| `ROCKSDB_LIB_DIR` | 自定义 RocksDB 库路径 | 系统路径 |
| `ROCKSDB_INCLUDE_DIR` | 自定义 RocksDB 头文件路径 | 系统路径 |
